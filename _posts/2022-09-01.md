## 문자열 뒤집기

문자열을 뒤집는 함수를 작성하라. 입력값은 문자 배열이며, 리터 없이 리스트 내부를 직접 조작하라.

```python
// 입력
["h","e","l","l","o"]

// 출력
["o","l","l","e","h"]


// 풀이
def reverseString(self, s: List[str]) -> None:
  s.reverse()
```
